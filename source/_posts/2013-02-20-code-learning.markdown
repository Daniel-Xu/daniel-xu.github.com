我的环境配置
------

1. 安装 `gemedit`
        gem install gemedit

2. 配置bash alias, 选择用我最熟悉的macvim来打开gem, 在`.bashrc`中加入如下：
        alias="gem edit -e mvim"
   打开的方式为：
        ge gemfile_name
